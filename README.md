# warehouse-robotic-simulation
Python tabanlı depo robotları simülasyonu. Projede depo ortamı grid yapısıyla modellenir, robotlara BFS rota planlama ile görev atanır ve farklı robot sayılarının sistem performansına etkisi analiz edilir. Tamamlanma süresi, throughput, robot kullanım oranı, bekleme süresi ve mesafe gibi metrikler CSV tabloları ve grafikler halinde üretilir.
